# Roblox NPC Dialogue System

A simple Roblox dialogue system built in Luau as a scripting practice project.

## Features

- NPC interaction using a ProximityPrompt
- Dialogue UI
- Multiple dialogue responses
- Typewriter text effect
- Clean, beginner-friendly code

## Purpose

This project was built to practice:

- LocalScripts
- GUI programming
- Event connections
- Typewriter effects
- Dialogue flow

## Notes

This is a focused scripting exercise rather than a complete game system. The goal was to learn and demonstrate Roblox UI and dialogue programming in a simple, easy-to-understand way.
